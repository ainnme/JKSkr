# JKSkr
一个基于jquery的快速开发框架<br>
官网:www.jkskr.com

说明:<br>
1.jkskr框架全部采用jquery的链式调用规则,所以只要你会jquery你就会jk,其他外部引用的js无需做更多操作,可以直接采用别人(你引用的其他JS)的规则
2.涵盖了市面上可以见到的大多数布局方式,让你可以做到拷贝粘贴的快捷开发,大数据布局不仅可以快速布局大数据分块,还能用来布局各种各样的WEB样式
3.jk将会不定期的更新新的模块功能，新的视觉效果，新的布局方式，让你的开发变得更有型更有趣。
4.jk内置了一个高端管理后台,摒弃常用的iframe嵌套,改用单页形式,支持F5刷新(即浏览器自带的刷新),刷新后之前访问的未关闭页面恢复,右侧导航卡可以设置显示多少个,超过设置将会从最开始的移除
